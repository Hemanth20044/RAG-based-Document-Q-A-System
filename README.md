RAG-based Document Q&A System
- Built a production-style retrieval-augmented generation pipeline using 
  LangChain, FAISS vector store, and Groq LLM (LLaMA 3)
- Implemented semantic chunking, embedding via sentence-transformers, 
  and conversational memory for multi-turn Q&A
- Deployed on Streamlit Cloud with source citation and chat history
Tech: Python, LangChain, FAISS, HuggingFace, Groq API, Streamlit
